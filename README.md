# Unova Quest (Gen 5 Inspired)

This repository now contains a **Pokémon Gen 5 inspired mini-game** built with **HTML, CSS, and vanilla JavaScript**.

## Why this language/stack?
I used browser-based JavaScript because it is the fastest way to deliver a playable game with no installation beyond a browser.

## Features
- Choose one starter: **Snivy, Tepig, or Oshawott**
- Move around a tile-based route with **WASD** or **Arrow keys**
- Trigger random wild encounters in tall grass
- Turn-based battles with multiple moves and healing moves
- Battle log and HP tracking

## Run locally
From the repo root:

```bash
python3 -m http.server 8000
```

Then open:

- `http://localhost:8000`

## Notes
This is a legal-safe, fan-inspired prototype and not an exact recreation of Pokémon Black/White.
